# Return the first n observations of a MiltSeries

Return the first n observations of a MiltSeries

## Usage

``` r
milt_head(series, n = 6L)
```

## Arguments

- series:

  A `MiltSeries` object.

- n:

  Number of observations to return. Default 6.

## Value

A `MiltSeries`.
