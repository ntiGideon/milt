# milt: Modern Integrated Library for Timeseries

A unified framework for time series forecasting, anomaly detection,
classification, and analysis in R. Provides a consistent API across
classical statistical models, machine learning, and deep learning
approaches. Includes backtesting, pipeline construction, multi-series
support, and model explainability.
