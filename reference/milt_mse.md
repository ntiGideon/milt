# Mean Squared Error

Mean Squared Error

## Usage

``` r
milt_mse(actual, predicted)
```

## Arguments

- actual:

  Numeric vector of observed values.

- predicted:

  Numeric vector of predicted values.

## Value

A single numeric value.
