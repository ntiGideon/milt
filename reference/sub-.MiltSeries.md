# Subset a MiltSeries by row index

Subset a MiltSeries by row index

## Usage

``` r
# S3 method for class 'MiltSeries'
x[i, ...]
```

## Arguments

- x:

  A `MiltSeries` object.

- i:

  Integer index, logical vector, or a length-2 Date/POSIXct vector
  specifying `c(start, end)`.

- ...:

  Ignored.
